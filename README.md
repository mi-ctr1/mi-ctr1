A Python-based AI workflow that classifies incoming requests using a large
language model. Given a user input, the system returns a structured response
including category, priority level, and a one-sentence summary — researching
what it takes to make LLM-driven automation consistent, observable, and
reliable under varied real-world inputs.

Concepts explored:
`Python` `Generative AI` `LLM` `Prompt Engineering` `AI Workflow Orchestration`
`Reliability` `Observability` `Automation`
